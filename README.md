# SelfCheck - 건강 자가진단

증상을 선택하여 가능한 질환을 확인할 수 있는 무료 온라인 자가진단 도구입니다. 감기부터 암까지 70여 가지 질환을 자가진단할 수 있습니다.

## 주요 기능

- **14개 증상 카테고리**: 호흡기, 소화기, 심혈관, 신경계, 근골격계, 피부, 비뇨/생식, 내분비/대사, 이비인후과, 안과, 감염성, 종양/암, 정신건강, 전신
- **87개 증상**: 각 증상 검색 가능
- **71개 질환 DB**: 감기부터 폐암, 뇌졸중, 각종 암까지 포함
- **4단계 진단 프로세스**: 카테고리 선택 → 증상 선택 → 증상 기간 → 결과 분석
- **매칭 알고리즘**: 증상 일치도, 위험도, 지속 기간을 종합 분석
- **개인정보 보호**: 모든 처리가 브라우저에서 이루어지며 서버로 데이터가 전송되지 않음

## 사용 방법

1. [index.html](index.html)을 브라우저로 엽니다.
2. 증상 카테고리(예: 호흡기, 소화기 등)를 선택합니다.
3. 해당 카테고리에서 겪고 있는 증상을 모두 선택합니다.
4. 증상이 지속된 기간을 선택합니다.
5. 진단 결과를 확인하고, 필요시 의료기관을 방문합니다.

서버나 설치가 필요 없는 단일 HTML 파일로 작동합니다.

## ⚠️ 의료 면책 공지

이 자가진단은 **참고용**이며 의학적 진단을 대체하지 않습니다. 정확한 진단을 위해 반드시 의사와 상담하시기 바랍니다. 응급 상황(심한 흉통, 호흡곤란, 편측 마비, 의식 저하 등)이 의심되면 **즉시 119에 연락**하세요.
